# Properties of Matrices

There are a number of functions available for determining certain properties of matrices. The functions are as follows.

<ul>
    <li>hasInvXmtx</li>
    <li>isDiagXmtx</li>
    <li>isIdtXmtx</li>
    <li>isLinIndXvec</li>
    <li>isOrthogonal</li>
    <li>isRdcFrmXmtx</li>
    <li>isRightHanded</li>
    <li>isSqrXmtx</li>
    <li>isSymXmtx</li>
</ul>
